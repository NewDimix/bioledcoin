Выкуп токенов LBC

  • По завершения этапа этапа 1.1. будет объявлен выкуп токенов с 140% маржой;
  • По завершения этапа этапа 1.2. будет объявлен выкуп токенов с 160% маржой;
  • По завершения этапа этапа 1.3. будет объявлен выкуп токенов с 200% маржой;
  • По завершения этапа этапа 2. будет объявлен выкуп токенов с 770% маржой.

После реализации всех этапов бизнес плана и всех выплат по проекту, компания предлагает для мотивации покупки токенов LBC на бирже, выплату дивидендов в размере 50% от чистой прибыли компании каждые 6 месяцев, прибыль будет выплачиваться пропорционально количеству токенов при условии, что токены будут находиться на кошельке платформы «Waves».
